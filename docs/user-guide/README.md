# Guia do usuário

- **Operação diária:** login com usuário fornecido pelo administrador; menus conforme perfil (RBAC).
- **Metadata Studio:** criação de entidades e campos dinâmicos — ver também `sidebar-nocode.md` nesta pasta.
- **Produção / estoque / vendas:** fluxos nas páginas dedicadas do menu lateral; dados persistidos na API (sem modo offline/mock).

Para credenciais de demo e subida do ambiente, use o **`README.md`** na raiz do repositório.
