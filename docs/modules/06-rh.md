# Módulo 6 – RH (Recursos Humanos)

## Objetivo
Gestão de funcionários, ponto eletrônico, solicitação de férias e folha de pagamento.

## Arquivos Backend

| Arquivo | Descrição |
|---------|-----------|
| `apps/backend/src/modules/hr/hr.module.ts` | Registra rotas |
| `apps/backend/src/modules/hr/hr.routes.ts` | Endpoints funcionários, ponto, férias, folha |
| `apps/backend/src/modules/hr/hr.service.ts` | Cálculo de horas, folha, aprovação de férias |

## Arquivos Frontend

| Arquivo | Descrição |
|---------|-----------|
| `apps/frontend/src/pages/rh/Funcionarios.jsx` | CRUD de funcionários |
| `apps/frontend/src/pages/rh/Ponto.jsx` | Registro manual de ponto + relatório de horas |
| `apps/frontend/src/pages/rh/Ferias.jsx` | Solicitação e aprovação de férias |
| `apps/frontend/src/pages/rh/FolhaPagamento.jsx` | Cálculo e exportação de folha |

## Endpoints

| Método | Rota | Descrição |
|--------|------|-----------|
| GET/POST | `/api/hr/employees` | CRUD funcionários |
| PATCH | `/api/hr/employees/:id` | Atualizar funcionário |
| GET/POST | `/api/hr/time-entries` | Registro de ponto (entrada/saída) |
| GET/POST | `/api/hr/leave-requests` | Solicitação de férias/licença |
| POST | `/api/hr/payroll/calculate` | Calcular folha do mês |
| GET | `/api/hr/payroll/:month` | Consultar folha (AAAA-MM) |

## Modelos Prisma

- `Employee` — userId, cargo, data admissão, salário base, carga horária
- `TimeEntry` — funcionário, entrada, saída, data, horas normais, extras
- `LeaveRequest` — funcionário, data início/fim, tipo (férias/licença), status
- `PayrollRun` — mês, ano, salário base, descontos, extras, líquido

## Cálculo de Folha

- **Horas extras** = horas acima de 44h semanais × fator 1,5 (configurável em `Parâmetros`)
- **Descontos**: INSS (tabela progressiva mock), IRRF (tabela mock), faltas
- **Líquido** = salário bruto + extras − descontos

## Permissões

`ver_rh`, `criar_rh`, `editar_rh`, `deletar_rh`, `ver_folha`

## Como Testar

1. Acesse **RH → Funcionários** e cadastre um funcionário vinculado a um usuário.
2. Registre entradas e saídas em **Ponto**.
3. Crie uma **Solicitação de Férias** e aprove.
4. Acesse **Folha de Pagamento**, selecione o mês e calcule.
